# ExZistence: EE Club @ UPM

Welcome to the official website repository for ExZistence, the Electrical Engineering Club at the University of Prince Mugrin.

## Overview
This project is a lightweight, responsive landing page designed to connect students with club resources, social media platforms, and membership applications. 

## Features
* **Dynamic Theme:** Built-in dark and light mode toggle that remembers user preferences.
* **Social Hub:** A centralized grid of all official club platforms that is easy to update.
* **Automated Announcements:** Time-sensitive banners that automatically appear and expire based on set dates, such as the All-Classes Competition.
* **Pure Front-End:** Built with standard HTML, CSS, and vanilla JavaScript for fast loading and easy hosting.

## Getting Started
To view or edit the website locally:
1. Clone or download this repository to your computer.
2. Open the index.html file directly in any modern web browser.
3. To update social links or event dates, edit the variables inside the scripts.js file.

## Hosting
This website is completely static and ready to be hosted for free using GitHub Pages. Simply push the code to your repository and enable Pages in the repository settings to make the club website live.
